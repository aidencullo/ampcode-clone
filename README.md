# Amp Website Clone

A replica of the Amp coding agent platform website, built from scratch with pure HTML, CSS, and JavaScript.

## Features

- Responsive design with modern UI/UX
- Animated elements and interactive effects
- Dark theme with gradient accents
- Smooth scrolling navigation
- Particle background effects
- Hover animations on cards and buttons
- Typing effect on hero subtitle
- Dynamic gradient on hero title

## Technology Stack

- HTML5
- CSS3 (with CSS variables and animations)
- Vanilla JavaScript (no frameworks)

## Sections

1. **Navigation** - Sticky header with product links and authentication buttons
2. **Hero** - Main headline with call-to-action
3. **Installation** - CLI/TUI and editor integration instructions
4. **Featured Threads** - Showcase of AI-assisted development projects
5. **News** - Latest platform updates and announcements
6. **Testimonials** - User feedback and reviews
7. **Footer** - Product links, resources, guides, and community links

## Deployment

This site is deployed on GitHub Pages.

## Local Development

Simply open `index.html` in a modern web browser. No build process required.
